# Jet

 
## Установка и сборка

Устанавливить зависимости сборщика и зависимости исходников, собрать исходники, устанавить зависимости собранного:
```
npm install
npm run-script install-source
npm run-script build
npm run-script install-app
```

 
## Запуск окружения

Собирает исходники и следит за изменениями:
```
npm run-script watch
```

 
## Запуск приложения

Запускает собранное приложение:
```
npm run-script serve
```
