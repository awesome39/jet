# Jet

 
## Установка

Устанавливает зависимости сборщика и зависимости исходников, собирает исходники, устанавливает зависимости собранного:
```
npm install
```

 
## Сборка

Собирает исходники:
```
npm run-script build
```

 
## Запуск окружения

Собирает исходники и следит за изменениями:
```
npm run-script build
```

 
## Запуск приложения

Запускает собранное приложение:
```
npm run-script serve
```
